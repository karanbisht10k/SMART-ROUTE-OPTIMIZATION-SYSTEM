# 🧠 Algorithms Used

## Dijkstra's Algorithm
- Guarantees the shortest path
- Explores every possible optimal route
- Best suited for weighted graphs

---

## A* Search Algorithm

- Faster than Dijkstra in most cases
- Uses heuristic-based searching
- Optimized for intelligent pathfinding
